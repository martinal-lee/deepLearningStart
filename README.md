# deepLearningStart

[English](./README_EN.md)

跟着《深度学习入门：基于Python的理论与实现》（鱼书）敲的代码，用 Jupyter Notebook 写的，边学边记。

**内容**

- `0_perceptron.ipynb` — 感知机
- `1_neural_network.ipynb` — 神经网络
- `2_neural_network_learning.ipynb` — 神经网络的学习
- `mnist/` — 手写数字数据集（训练集 + 测试集）

**环境**

Python + Jupyter Notebook，依赖标准深度学习库（numpy 等）。
